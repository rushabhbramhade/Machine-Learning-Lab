## Suggested Order (don't go top to bottom blindly)
1. Python → NumPy → Pandas → SQL (tools)
2. Statistics (theory you need before ML makes sense)
3. EDA → Feature Engineering
4. Machine Learning (classical algorithms)
5. Deep Learning (fundamentals)
6. MLOps (deployment skills — this is what makes you "Engineer" not just "Scientist")
7. Pick ONE specialization first: NLP, Computer Vision, Time Series, or LLM Engineering. Don't try to do all four in parallel.
